# Calendar Usage Instructions

## Navigating the Calendar

### Week Navigation
- Use **← →** arrows at the top to switch between weeks
- Current week range is displayed in the header

### Day Selection
- Click on any **day tab** (Sun, Mon, Tue, etc.) to view that day's schedule
- Selected day is highlighted in **cyan**
- Each tab shows the day name and date number

## Viewing Events

### Daily View
- Events are displayed in **hourly time slots** (9:00 AM - 6:00 PM)
- Each event shows:
  - **Title** (bold text)
  - **Description** (if available)
  - **Location** (with 📍 icon, if available)

### Empty Time Slots
- Time slots without events show *"No events"* in gray text
- Perfect for planning new activities

## Navigation Tips

- **Mobile-Friendly**: Scroll vertically through your day
- **Week Overview**: Swipe through day tabs to see the whole week
- **Quick Access**: Header always visible for easy navigation

## Next Steps

- To add events, use the **Import** feature
- Sign out using the **⊗ SIGN OUT** button in the header
