# Import Calendar Instructions

## Importing Your Calendar

### Step 1: Export from Your Calendar App

#### Google Calendar
1. Open Google Calendar on desktop
2. Click **Settings** (gear icon) → **Settings**
3. Select **Import & Export** from left sidebar
4. Click **Export**
5. Download the `.zip` file
6. Extract to find `.ics` files

#### Apple Calendar
1. Open Calendar app on Mac
2. Select the calendar you want to export
3. Go to **File** → **Export** → **Export...**
4. Save the `.ics` file to your computer

#### Outlook
1. Open Outlook
2. Go to **File** → **Save Calendar**
3. Choose **iCalendar Format (.ics)**
4. Save the file

### Step 2: Upload to Scheduler

1. **Navigate to Import Page**
   - Click **"Import"** in the header navigation
   - Or go to `/calendar/import`

2. **Select File**
   - Click the upload area or drag-and-drop your `.ics` file
   - File name and size will be displayed

3. **Import**
   - Click **"⊕ IMPORT CALENDAR"** button
   - Wait for processing (usually a few seconds)

4. **Confirmation**
   - Success message shows number of events imported
   - Events are now visible in your calendar

## Important Notes

- **File Format**: Only `.ics` files are supported
- **Data Privacy**: Your calendar data stays on our server, not shared with Google/Apple
- **Duplicates**: The system skips duplicate events automatically
- **No Account Linking**: You don't need to connect your Google/Apple account

## Troubleshooting

- **Import Failed**: Check that file is valid `.ics` format
- **No Events Showing**: Make sure events are within the current week view
- **Large Files**: May take longer to process (be patient)

## After Import

- Navigate to **Calendar** to view your imported events
- Events are organized by day and time
- Use week navigation to browse all imported events
